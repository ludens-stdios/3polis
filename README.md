## ✨ Overview

**3POLIS** is a city-building simulation game developed with **Three.js** and **TypeScript**, focused on creating a performant, modular and extensible simulation engine for the browser.

The project aims to combine:

- 🏙️ Rich city-building mechanics
- 🚗 Traffic simulation
- ⚡ Resource management
- 👥 Citizen behaviour
- 📈 Dynamic economy
- 🌍 Large-scale procedural worlds

Everything runs directly in the browser using WebGL.

---

# 🚀 Features

- 🌆 Modern 3D rendering with Three.js
- 🧩 Modular ECS-inspired architecture
- 🛣️ Road placement tools
- 🏠 Zone management
- 🚦 Vehicle pathfinding
- 💡 Resource simulation
- 🌳 Environment generation
- 🎮 Smooth camera controls
- ⚙️ TypeScript-first development

```bash
npm install

npm run dev
```
